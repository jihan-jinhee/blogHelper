# blogHelper

## 네이버 블로그 활동을 보조하는 목적으로 만든 프로그램
![image](https://user-images.githubusercontent.com/74603608/216958726-9dbee74b-0ed8-471d-b428-d601ae890ff5.png)

### 기능
* 사진에 워터마크 추가 (본인 전용. 타 워터마크 지원 X)
* 댓글 확인
* 이웃 글 방문
* 글자수세기
* 미방문이웃 확인 & 삭제
* 이웃 추가
